# projects-on-list-in-HTML

This repository contains practice exercises focused on HTML lists, created while learning web development from a beginner level, but structured using correct and advanced HTML semantics.
Although the exercises are beginner-friendly, the code follows best practices, making this repository useful for revision, reference, and practice even at a more advanced level.

This repository is ideal for:
Beginners learning HTML
Students revising lists before exams
Anyone confused about nested lists
Developers who want clean reference examples
