THE THREE PILLARS OF WEB DEVELOPMENT

    1.HTML
    HYPERTEXT MARKUP LANGUAGE

    Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser

    2.CSS
    CASCADING STYLE SHEETS

    CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML or XML

    3.JS
    JAVASCRIPT

    JavaScript is a versatile, high-level programming language that powers the behavior of web pages, making them interactive and dynamic
